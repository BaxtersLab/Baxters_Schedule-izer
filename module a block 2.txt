PACKAGE MAP
--------------------------------------------------------------------------------

Root package: com.baxter.schedulaizer

  com.baxter.schedulaizer
    └── SchedulaizerApp.kt
          Application subclass. Initializes the Room database singleton and
          provides global access to repositories.

  com.baxter.schedulaizer.data.db
    └── SchedulaizerDatabase.kt
          @Database annotated Room class. Lists all entity classes. Exposes
          DAO accessors. Version 1 with migration stubs for future versions.
    └── TypeConverters.kt
          Room @TypeConverter functions for Long<->Date and List<->String.

  com.baxter.schedulaizer.data.db.entity
    └── EventEntity, BillEntity, MemoEntity, AttachmentEntity, AlertEntity
          @Entity annotated data classes. One per table.

  com.baxter.schedulaizer.data.db.dao
    └── EventDao, BillDao, MemoDao, AttachmentDao, AlertDao
          @Dao annotated interfaces. All read queries return Flow<List<T>>.
          Write operations are suspend functions.

  com.baxter.schedulaizer.data.repository
    └── EventRepository, BillRepository, AttachmentRepository
          Plain classes (not interfaces). Injected via SchedulaizerApp.
          Contain business logic: sorting, filtering, next-due calculation.
          Expose StateFlow / Flow to ViewModels.

  com.baxter.schedulaizer.util
    └── PriorityLevel.kt    — enum class
    └── DateUtils.kt        — object singleton
    └── FileUtils.kt        — object singleton

  com.baxter.schedulaizer.service
    └── SchedulaizerService.kt
          Foreground Service. Manages tunnel session lifecycle. Starts on boot.

  com.baxter.schedulaizer.alerts
    └── AlertScheduler.kt          — schedules WorkManager + AlarmManager items
    └── AlertReceiver.kt           — BroadcastReceiver for exact alarms
    └── AlertNotificationBuilder.kt — builds NotificationCompat objects

  com.baxter.schedulaizer.transfer
    └── FileTransferManager.kt     — queues and sends files via RD tunnel
    └── BudgetBlasterBridge.kt     — packages receipt payloads, HTTP POST

  com.baxter.schedulaizer.voice
    └── VoiceListenerService.kt    — wraps Android SpeechRecognizer
    └── IntentParser.kt            — maps utterances to SchedulerIntent
    └── AgentDispatcher.kt         — routes intents to repository actions

  com.baxter.schedulaizer.ui
    └── MainActivity.kt            — BottomNavigationView host activity
    └── calendar/CalendarFragment.kt + CalendarViewModel.kt + EventEditActivity.kt
    └── bills/BillsFragment.kt + BillsViewModel.kt + BillEditActivity.kt
    └── capture/CaptureFragment.kt + CaptureViewModel.kt
    └── transfer/TransferFragment.kt + TransferViewModel.kt
    └── agent/AgentFragment.kt + AgentViewModel.kt
